# Abroad Compass

Your complete Germany study-abroad operating system. Plan your German master's application end-to-end: university shortlisting, German learning, budget, documents, and an AI mentor.

## Features

- **AI Mentor**: AI assistance for reviewing SOPs, generating recommendations, and answering questions. (Powered by Google / OpenRouter)
- **University Shortlisting**: Keep track of the universities you are applying to.
- **Budget Planning**: Manage and forecast your expenses.
- **Document Management**: Manage CVs, SOPs, LORs, and transcripts.
- **German Learning Module**: Monitor your language learning progress.

## Tech Stack

- **Frontend**: React 19, TanStack Start (SSR/Router/Query), Tailwind CSS, Radix UI.
- **Backend**: Supabase (PostgreSQL, Auth, RLS, Storage).
- **AI Integration**: AI SDK with OpenAI-compatible gateway pointing to Google/OpenRouter models.
